## 📋 المتغيرات البيئية المطلوبة

قم بإنشاء ملف `.env` في مجلد `backend/` مع هذه المتغيرات:

```env
# أساسي
SECRET_KEY=<generate-with-django>
DEBUG=False
ALLOWED_HOSTS=your-domain.com,www.your-domain.com

# قاعدة البيانات
DB_NAME=wasel_water_db
DB_USER=wasel_user
DB_PASSWORD=<strong-password>
DB_HOST=localhost
DB_PORT=5432

# CORS
CORS_ALLOWED_ORIGINS=https://your-frontend.com

# الأمان
SECURE_SSL_REDIRECT=True
SESSION_COOKIE_SECURE=True
CSRF_COOKIE_SECURE=True

# API الخارجية
GOOGLE_MAPS_API_KEY=<your-key>
```

## ⚠️ نقاط مهمة جداً

1. **لا تنسى تغيير DEBUG إلى False**
   - يجب `DEBUG=False` في ملف `.env`
   
2. **استخدم PostgreSQL وليس SQLite**
   - SQLite غير آمن للإنتاج

3. **احصل على SSL Certificate**
   - استخدم Let's Encrypt (مجاني)

4. **استخدم Nginx كـ Reverse Proxy**
   - لا تشغّل Django مباشرة على المنافذ

5. **عمل Backups منتظم**
   - للبيانات والملفات المرفوعة
