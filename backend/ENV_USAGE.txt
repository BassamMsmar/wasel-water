# 🎯 Correct .env File Usage Instructions

## ✅ Correct Steps

### 1️⃣ Copy File
```bash
cp .env.production .env
```

### 2️⃣ Edit Variables in .env

```env
SECRET_KEY=your-actual-secret-key
DEBUG=False
ALLOWED_HOSTS=your-domain.com
DB_NAME=wasel_water_db
DB_USER=wasel_user
DB_PASSWORD=strong-password
DB_HOST=localhost
DB_PORT=5432
CORS_ALLOWED_ORIGINS=https://your-frontend.com
```

### 3️⃣ Test Load
```bash
python manage.py shell
>>> from dotenv import load_dotenv
>>> load_dotenv()
>>> print("✅ Loaded successfully!")
```

### 4️⃣ شغّل التطبيق
```bash
python manage.py check --deploy
python manage.py runserver
```

## ❌ تجنب هذه الأخطاء

### ❌ خطأ 1: مسافات حول `=`
```env
SECRET_KEY = value    # ❌ خطأ (مسافات)
SECRET_KEY=value      # ✅ صحيح (بدون مسافات)
```

### ❌ خطأ 2: علامات اقتباس
```env
SECRET_KEY="value"    # ❌ قد يسبب مشاكل
SECRET_KEY=value      # ✅ صحيح (بدون علامات)
```

### ❌ خطأ 3: تعليقات معقدة
```env
# Don't use 'quotes' in comments  # ❌ قد يسبب مشاكل
# Use simple comments              # ✅ صحيح
```

### ❌ خطأ 4: أحرف خاصة
```env
PASSWORD=my@pass&123!@#   # ⚠️ قد يحتاج escape
PASSWORD=mypass123        # ✅ بسيط وآمن
```
